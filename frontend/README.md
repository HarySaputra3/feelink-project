### Prerequisites

- [Node.js (v16+)](https://nodejs.org/)
- npm (comes with Node.js) or use [pnpm](https://pnpm.io/) / [yarn](https://yarnpkg.com/)

---

### Installation

```
# Navigate to the frontend folder
cd frontend

# Install dependencies
npm install
```

### Run locally
```
npm run dev
```

### Built with
- Vite
- React
- Tailwind CSS
